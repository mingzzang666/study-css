## 🎨 CSS 요약

### ✅ CSS란?
- **Cascading Style Sheets**의 약자  
- HTML 요소가 다양한 **미디어(화면, 인쇄 등)** 에서 어떻게 보이는지를 정의  
- **스타일을 HTML 구조로부터 분리**하여 유지보수성과 재사용성을 높임  

---

### 💡 CSS를 사용하는 이유
- HTML만 사용할 경우, **각 요소에 직접 스타일 지정**해야 해서 비효율적  
- **W3C 표준 언어인 CSS**로 스타일을 한 곳에서 제어 가능  
- 스타일 일관성 유지, 수정/유지보수 용이  
- **외부 파일로 관리**하면 사이트 전체 스타일을 쉽게 변경 가능  

---

### 🧩 기본 문법
```css
선택자 {
    속성명: 속성값;
}

/* 예시 */
p {
    text-align: center;
    color: blue;
}
```

---

### 🧭 선택자 종류
1. **전체 선택자 (`*`)**  
   - 모든 요소에 공통 스타일 적용  

2. **태그 선택자 (`p`, `h1` 등)**  
   - 특정 태그 이름의 모든 요소에 적용  

3. **클래스 선택자 (`.className`)**  
   - 동일한 클래스 이름을 가진 여러 요소에 스타일 적용  

4. **아이디 선택자 (`#idName`)**  
   - 특정 요소 하나에만 스타일 적용  
   - 문서 내 중복 불가  

5. **그룹 선택자 (`h1, h2, p`)**  
   - 여러 선택자에 동일한 스타일을 한 번에 적용  

---

### ⚖️ 캐스케이딩(Cascading) — 우선순위 규칙
#### 1️⃣ 중요도 (선언 위치)
1. 인라인 스타일 (`<div style="">`)
2. 내부 스타일 시트 (`<style>`)
3. 외부 스타일 시트 (`.css 파일`)
4. 브라우저 기본 스타일  

#### 2️⃣ 명시도 (선택자 구체성)
1. `!important`
2. 아이디 선택자 (`#id`)
3. 클래스 선택자 (`.class`)
4. 태그 선택자 (`div`)
5. 전체 선택자 (`*`)
6. 상속받은 속성  

---

### 🧱 정렬 기본
#### 🔹 블록 요소
- **가로 정렬(본인)**: `margin: 0 auto;`  
- **자식 정렬 (flex 사용 시)**  
  - 기본: 가로(`row`) 배치  
  - `flex-direction: column` 시 세로 배치로 전환  
  - 수평 정렬: `justify-content`  
  - 수직 정렬: `align-items`  

#### 🔹 인라인 요소
- **수평 정렬(부모)**: `text-align`  
- **수직 정렬(본인)**: `line-height`
